Galicia 2025 — Pack de Deploy (.bat) — Configurado para Ring8-web
=================================================================

Este pack ya está listo para tu repositorio:
👉 https://github.com/Ring8-web/galicia2025.git

Incluye:
- deploy.bat              → git add/commit/push
- deploy-with-build.bat   → build + push
- .gitignore              → ignora node_modules, dist, etc.

Cómo usar:
-----------
1. Copia los archivos en la raíz de tu proyecto.
2. Doble clic en deploy.bat (o deploy-with-build.bat).
   - El script hará commit y push al repo https://github.com/Ring8-web/galicia2025.git
   - Vercel se encargará del deploy automáticamente.
3. Espera 1–2 minutos y abre tu sitio en producción:
   🌐 https://galicia2025.vercel.app
